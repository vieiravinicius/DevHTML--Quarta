Bem-vindos a Homepage do CloudFlix: Catálogo de Filmes

O CloudFlix é um projeto de demonstração criado para ilustrar os conceitos fundamentais do desenvolvimento web. Pense nele como o seu cinema digital pessoal. Mas, em vez de simplesmente assistir aos filmes, você vai entender como ele funciona por dentro.

Neste projeto, você aprenderá sobre:

  • Front-End: A parte visual do site, que você está vendo agora. Como os filmes são organizados e exibidos, usando HTML, CSS e JavaScript.
  • Back-End: O "cérebro" do site. É aqui que os dados dos filmes são armazenados e gerenciados.
  • Banco de Dados: A nossa biblioteca de filmes. Todos os títulos, sinopses e imagens que você vê estão guardados aqui.
O nome CloudFlix não é por acaso. Ele representa a união do `Cloud` (nuvem), que simboliza a infraestrutura de servidores onde o projeto está hospedado, com o sufixo `-flix`, um ícone dos serviços de streaming. Em resumo, este projeto é a sua oportunidade de colocar a mão na massa, entender a estrutura de uma aplicação web real e transformar a teoria strong prática. Explore o catálogo, entenda cada parte do código e prepare-se para criar seu próprio projeto! Este é o conteúdo principal da página de início. Você pode navegar para outras páginas usando os links acima. Este projeto foi criado apenas com HTML, demonstrando a estrutura básica de um site.
